## 🎯 Cách Sử Dụng

### Bước 1: Quét Job
```
1. Click "🔍 Scan"
2. Hệ thống tìm job mới
3. Danh sách cập nhật
```

### Bước 2: Xem Loại Project
```
1. Chọn job từ danh sách
2. Xem panel giữa
3. Kiểm tra "🏆 Cuộc Thi" hoặc "💼 Bid Trực Tiếp"
```

### Bước 3: Apply
```
Nếu là CUỘC THI:
  1. Click "🏆 Tham Gia Cuộc Thi"
  2. Xác nhận
  3. Entry được submit
  
Nếu là BID TRỰC TIẾP:
  1. Click "✅ Gửi Đề Xuất"
  2. Xác nhận
  3. Proposal được gửi
```

### Bước 4: Chờ Kết Quả
```
Hệ thống giám sát tự động:
- Phát hiện khi khách hàng liên hệ
- Phát hiện khi được trao giải
- Thông báo real-time
```

## 💰 Bid Strategy Mới

```
Job Budget: $100

```

### Sau (Bid Cao Hơn 15%)

### Ví Dụ
```
$50 job → Bid $57.50
$100 job → Bid $115
$200 job → Bid $230
$500 job → Bid $575
```

## 🏆 Cuộc Thi vs Bid Trực Tiếp

### Cuộc Thi (🏆)
```
Cách hoạt động:
- Bạn submit entry
- Khách hàng xem nhiều entry
- Khách hàng chọn entry tốt nhất
- Bạn nhận tiền

Lợi ích:
✅ Cơ hội thắng cao hơn
✅ Khách hàng chọn quality tốt
✅ Tránh job bị xoá
✅ Thu nhập cao hơn

Rủi ro:
⚠️ Có thể không thắng
⚠️ Phải submit entry
```

### Bid Trực Tiếp (💼)
```
Cách hoạt động:
- Bạn gửi proposal
- Khách hàng chọn 1 freelancer
- Bạn bắt đầu làm việc
- Bạn nhận tiền

Lợi ích:
✅ Nhanh chóng
✅ Rõ ràng
✅ Dễ quản lý

Rủi ro:
⚠️ Cạnh tranh nhiều
⚠️ Job có thể bị xoá
```

## 📊 Phát Hiện Tự Động

### Hệ Thống Phát Hiện
```
Tìm keyword:
- "contest" → Cuộc thi
- "competition" → Cuộc thi
- "entries" → Cuộc thi
- "winner" → Cuộc thi
- "choose best" → Cuộc thi

Nếu tìm thấy → 🏆 Cuộc Thi
Nếu không → 💼 Bid Trực Tiếp
```

### Ví Dụ
```
"Logo Design Contest - Submit 3 entries, I'll choose the best"
→ 🏆 Cuộc Thi

"Logo Design - Fixed Price $100"
→ 💼 Bid Trực Tiếp
```

```

```
